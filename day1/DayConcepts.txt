# Rest parameters
ES6 provides a new kind of parameters called rest parameters hat has a prefix of three dots
a rest parameters allows you represent an indefinite number of arguments as an array
the last parameters is prefix with ... 

# spread opeartor
Es6 provides a new opeartor called spread opeartor that consist of three dots
the spread opeartor allows you spread out element of an iterable object such array, map, set



# destructuring 
ES6 provide a new features called destructuring assignment that allow you to destructure
